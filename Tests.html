Multiply Tests

  • Expect Multiply(x*y) to create a product
  • Expect Multiply(5*2) to = 10
  • Expect Multiply("a"*2) to give an error
  1. When user inputs 2 numbers it should multiply them and get a product
  2. When user inputs more than 2 numbers it should multiply all the numbers together and get a product
  3. When user inputs a string of letters it should give an error
  4. When user inputs only 1 number it is multiplied by itself
  5. When user inputs nothing it should give an error

concatOdds Tests

  • Expect concatOdds to take 2 arrays of numbers and make them into 1 array of odd numbers in ascending order
  • Expect code to recognize negative numbers
  1. When user inputs repeated numbers in the arrays, the final array should only produce 1 number of those repeated
  2. When user inputs a "string" or "letters", the final array will only input the numbers found within the "string/letters"
  3. When user inputs numbers in the first 2 arrays, only odd numbers in a ascending order should result in the final array
  4. When user inputs a "string" of numbers, it should convert that string into numbers
  5. When user only inputs a "string" of letters, it should do nothing
